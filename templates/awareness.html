Phishing Awareness Guide

What is Phishing?

Phishing is a cybercrime where targets are contacted by email, telephone or text message by someone posing as a legitimate institution.

Common Phishing Signs
URL Red Flags
  • Misspelled domain names
  • Unusual subdomains
  • IP addresses instead of domains
Content Red Flags
  • Urgent action required
  • Grammar and spelling errors
  • Requests for personal information
Technical Red Flags
  • No HTTPS encryption
  • Suspicious attachments
  • Unusual sender addresses
